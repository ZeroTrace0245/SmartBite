🎯 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

1. User opens app and selects a pathway (GYM/DIET/WELLNESS)
2. The pathwayTheme is saved and passed to ALL screens
3. Each screen extracts the gradient colors:
   - gradientStart (primary color)
   - gradientEnd (secondary blue color)
4. All buttons and cards apply the theme colors
5. All animations use the theme colors
6. When user switches pathway, ALL screens update instantly!

🔄 PATHWAY THEME FLOW
═══════════════════════════════════════════════════════════════════════════════

1. User selects Gym pathway in PathwayPreferencesScreen
2. onThemeChosen(GYM) is called
3. MainActivity updates selectedTheme to GYM
4. SmartBitTheme updates and recomposes with GYM colors
5. ALL screens receive currentTheme = GYM
6. Every screen extracts:
   - gradientStart = GymGradientStart (Red)
   - gradientEnd = GymGradientEnd (Blue)
7. ALL buttons and cards apply Red/Blue colors
8. User navigates between screens - all have red theme!
9. If user switches to Diet - step 1-7 repeat with Green colors
10. Entire app updates instantly!
