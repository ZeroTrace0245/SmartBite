💾 TOKEN STORAGE FLOW:

FIRST ENTRY:
User enters token in dialog
         ↓
ChatViewModel.setToken(token)
         ↓
TokenManager.saveToken(token)
         ↓
SharedPreferences stores token
         ↓
Token saved on device permanently!

AUTO-LOADING:
ChatScreen launches
         ↓
ChatViewModel.init() runs
         ↓
tokenManager.hasToken() checked
         ↓
hasToken = true if token exists
         ↓
No dialog shown - ready to chat!

SENDING MESSAGE:
User types message
         ↓
User clicks Send
         ↓
ChatViewModel.sendMessage(messageText)
         ↓
Gets token: tokenManager.getToken()
         ↓
Sends to GitHub Models API
         ↓
AI responds with stored token!
