🔐 HOW TO GET YOUR GITHUB TOKEN:

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Name: "SmartBite AI"
4. Scopes: Check "read:user" and "user:email"
5. Copy the token
6. In app: Click AI Chat → Enter token in dialog

⚠️ IMPORTANT:
   • Keep your token SECRET
   • Never commit it to version control
   • You can regenerate tokens anytime

1. User opens AI Chat screen
2. Token dialog appears (first time)

6. Real AI responses appear in chat

8. All theme colors applied automatically

🔄 CONVERSATION FLOW:

User: "What should I eat after a workout?"

↓ ChatViewModel receives message & token

↓ GitHubModelsRepository creates request with:
  - Model: "gpt-4o-mini"
  - Messages: [system_prompt, prev_messages..., user_message]
  - Temperature: 0.7
  - Max tokens: 1024

↓ Retrofit sends to:
  https://models.inference.ai.azure.com/openai/deployments/gpt-4o-mini/chat/completions

↓ GitHub Models API processes

↓ Response returned with AI answer

↓ UI updates with response
