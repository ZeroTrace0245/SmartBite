CUSTOMIZATION QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

Want to customize? Here are the quick changes:

Change glow speed (DashboardScreen.kt):
  tween(1500)  → tween(1000)  = Faster
  tween(1500)  → tween(2000)  = Slower

Change glow intensity:
  initialValue = 0.3f → 0.1f  = Dimmer
  targetValue = 0.8f  → 0.95f = Brighter

Change shadow size:
  elevation = 12.dp → 20.dp = Bigger

Change border thickness:
  width = 1.5.dp → 3.dp = Thicker

Change colors (Color.kt):
  GymGradientStart = Color(0xFFYourColor)
