## 🔧 TROUBLESHOOTING

### If colors don't look right
→ Clean build: `./gradlew clean`
→ Rebuild: `./gradlew assembleDebug`

### If animations stutter
→ Test on different devices
→ Check device performance settings

### If text is hard to read
→ Check brightness settings
→ Verify contrast ratio (should be 12.6:1)

### If buttons look wrong
→ Verify 56dp height
→ Check 16dp corner radius
→ Ensure white text
