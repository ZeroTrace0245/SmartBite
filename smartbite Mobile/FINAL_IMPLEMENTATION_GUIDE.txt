🎓 CUSTOMIZATION GUIDE

To modify animations:

1. Change glow speed:
   tween(1500) → tween(1000)  // Faster
   tween(1500) → tween(2000)  // Slower

2. Change glow intensity:
   initialValue = 0.3f → 0.1f  // Dimmer when off
   targetValue = 0.8f → 0.95f  // Brighter when on

3. Change shadow size:
   elevation = 12.dp → 20.dp  // Bigger shadow

4. Change border thickness:
   width = 1.5.dp → 3.dp  // Thicker border

5. Change colors:
   Edit Color.kt - Update the gradient colors
